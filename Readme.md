# 🌈 Happy Holi - Interactive Festival Webpage

A beautiful and interactive webpage celebrating the Festival of Colors with vibrant animations and engaging features!

## ✨ Features

- **🎨 Animated Gradient Background** - Smooth color transitions with vibrant Holi colors
- **💫 Pichkari Cursor** - Custom cursor that changes colors every 2 seconds
- **🎯 Interactive Color Splash** - Click anywhere to create colorful bursts
- **🧭 Beautiful Navbar** - Gradient navigation bar with hover effects
- **📱 Responsive Footer** - Colorful footer with social icons
- **🎭 Floating Animations** - Smooth text animations

## 🚀 Live Demo

Simply open `index.html` in your browser to experience the magic!

## 🎮 How to Use

1. Move your mouse to see the colorful Pichkari cursor
2. Click anywhere on the page to splash colors
3. Watch the cursor change colors automatically every 2 seconds
4. Enjoy the animated gradient background

## 🛠️ Technologies Used

- HTML5
- CSS3 (Animations, Gradients, Flexbox)
- Vanilla JavaScript
- Google Fonts (Pacifico & Montserrat)
